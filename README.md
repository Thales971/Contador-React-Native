# Contador React Native

Este é um aplicativo simples de contador desenvolvido com **React Native** e **Expo**. O projeto demonstra o uso de hooks (`useState`), componentes básicos (`View`, `Text`, `TouchableOpacity`) e estilos com `StyleSheet`.

## Funcionalidades

- Incrementar e decrementar um valor
- Botões com cores distintas (verde para aumentar, vermelho para diminuir)
- Interface responsiva e minimalista

## Como executar

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicialize o servidor do Expo:
   ```bash
   npm start
   ```
3. Use o aplicativo Expo Go no seu dispositivo móvel ou um emulador para visualizar o projeto.
